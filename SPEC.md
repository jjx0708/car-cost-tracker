# 🚗 私家车用车成本统计 - 产品设计文档

## 1. 项目概述

### 1.1 产品定位
一款面向私家车主的用车成本记录与统计分析工具，帮助车主清晰了解车辆使用成本。

### 1.2 目标用户
- 私家车车主（尤其新能源车主）
- 想要精打细算控制用车成本的用户
- 需要记录和分析用车开销的用户

### 1.3 核心价值
- 记录每一次加油/充电，统计百公里能耗
- 汇总所有用车花费，生成周期性报表
- 数据可视化，了解支出结构和趋势

## 2. 功能清单

### 2.1 车辆管理
| 功能 | 描述 |
|------|------|
| 车辆信息 | 车型、购车日期、裸车价格 |
| 车辆切换 | 支持多辆车管理 |

### 2.2 能源记录
| 功能 | 描述 |
|------|------|
| 充电记录 | 金额、度数、日期、充电类型（家充/公充）、当前总里程 |
| 加油记录 | 金额、升数、单价、日期、加油站、当前总里程 |
| 能耗计算 | 自动计算两次补能间的百公里电耗/油耗 |

### 2.3 保养维护
| 功能 | 描述 |
|------|------|
| 保养记录 | 小保养、大保养、维修、配件更换 |
| 保养提醒 | 里程或时间提醒（可选功能） |

### 2.4 保险费用
| 功能 | 描述 |
|------|------|
| 交强险 | 金额、起止日期 |
| 商业险 | 金额、起止日期 |

### 2.5 日常花费 + 违章
| 功能 | 描述 |
|------|------|
| 洗车 | 金额、日期 |
| 停车 | 金额、日期，地点 |
| 过路费 | 金额、日期、路段 |
| 违章 | 金额、日期、原因、扣分 |

### 2.6 罚单统计
| 功能 | 描述 |
|------|------|
| 违章记录 | 罚款金额、扣分、日期、违章原因、处理状态 |

### 2.7 其他支出
- 车载用品
- 车内装饰
- 其他杂费

### 2.8 统计报表
| 维度 | 描述 |
|------|------|
| 周期统计 | 每周、每月、每年、总计花费 |
| 分类统计 | 能源、保险、保养、日常等占比 |
| 能耗统计 | 百公里电耗/油耗、平均单次花费 |
| 趋势图表 | 折线图展示月度花费趋势 |

## 3. 数据结构

### 3.1 车辆表 (vehicles)
```json
{
  "id": "uuid",
  "name": "车型名称",
  "purchaseDate": "2024-01-01",
  "purchasePrice": 150000,
  "initialMileage": 0,
  "createdAt": "timestamp"
}
```

### 3.2 能源记录表 (energy_records)
```json
{
  "id": "uuid",
  "vehicleId": "uuid",
  "type": "charge | fuel",
  "amount": 50.00,        // 金额
  "quantity": 20.0,       // 充电度数 或 加油升数

  "mileage": 15000,       // 记录时总里程
  "note": "备注",
  "createdAt": "timestamp"

```

## 4. 页面结构

### 4.1 页面清单
1. **首页/仪表盘** - 当月花费概览、快速记录入口
2. **记录页** - 添加各类记录
3. **统计页** - 报表和图表
4. **车辆页** - 车辆管理
5. **设置页** - 数据导出/导入、主题设置

### 4.2 导航设计
- 底部 Tab 导航：首页 | 记录 | 统计 | 我的
- 首页：车辆选择 + 当月概览 + 最近记录
- 记录：分类 Tab（能源/保养/保险/日常)
- 统计：时间筛选 + 图表 + 明细

---

## 5. 技术方案

### 5.1 前端
- **框架**：纯 HTML + CSS + JavaScript
- **存储**：LocalStorage（数据保存在浏览器本地）
- **图表**：Chart.js
- **部署**：GitHub Pages

### 5.2 数据存储
- 使用 LocalStorage 存储所有数据
- 支持数据导出为 JSON 文件
- 支持数据导入

## 6. 界面设计

### 6.1 风格
- 简洁现代，深色/浅色主题可选
- 移动端优先设计

### 6.2 配色
- 主色：#007AFF (蓝色)
- 背景：#F2F2F7 (浅色) / #000000 (深色)
- 强调色：#34C759 (绿色，用于收入/节省)
