# 用车成本统计 - 开发任务

## 项目概述
开发一个私家车用车成本统计的网页应用，功能包括：
- 车辆管理
- 能源记录（充电/加油）+ 能耗计算
- 保养、保险、日常花费、违章记录
- 统计报表（周/月/年/总计）

## 技术方案
- 纯 HTML + CSS + JavaScript
- LocalStorage 存储
- Chart.js 图表
- 单页应用，底部 Tab 导航
- PWA 配置

## 文件结构

## 核心功能

### 1. 车辆管理
- 添加车辆（车型、购车日期、价格、初始里程）
- 查看/编辑/删除车辆
- 车辆切换

### 2. 能源记录
- 充电记录：金额、度数、日期、类型（家充/公充）、里程
- 加油记录：金额、升数、日期、加油站、里程
- 自动计算百公里能耗

### 3. 其他记录
- 保养记录
- 保险记录
- 日常花费（洗车、停车、过路费）
- 违章记录（罚款、扣分、原因）

### 4. 统计报表
- 当月/本年/总计花费
- 分类占比
- 能耗统计

## UI 要求
- 移动端优先
- 底部 Tab 导航
- 表单弹窗输入
- 简洁现代风格
